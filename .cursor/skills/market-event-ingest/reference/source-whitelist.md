# 优先信息源（白名单倾向）

检索时应优先这些来源；仍须 ≥2 源交叉验证。

## 政策 / 宏观

- 美联储 / 各国央行官网与新闻稿
- 财政部、国会立法文本
- FRED / BEA / BLS 官方发布说明
- Wikipedia（仅作时间线入口，关键事实须一手源）

## 公司

- 公司 Investor Relations / 新闻稿
- SEC EDGAR（**已由站内 stockEvents 覆盖的 10-Q/10-K/8-K 勿再入库**）
- 交易所公告

## 评级 / 目标价

- 券商公开研究报告摘要页
- 可信财经汇总（注明机构名与日期）
- **禁止**无出处编造 Buy/Sell 或目标价数字

## 讲话

- 官方讲稿、听证会实录、公司活动转写
