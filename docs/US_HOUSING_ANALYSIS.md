# 美国住房与地产分析框架

本文档与宏观页内置模板、`housingAnalysisLayout.ts`、`.cursor/prompts/us-housing-analysis-framework.md` 保持一致。
Spec 与接入记录见 [specs/us-housing.spec.md](./specs/us-housing.spec.md)。

## 核心问题（L0）

> 房地产周期处于扩张、见顶还是收缩？利率对购房需求的压制走到哪一步？供给（开工/库存）与价格是否背离？作为利率最敏感、领先整体经济 2–4 季度的部门，住房是拖累还是支撑？

## 两模板链条（宏观页 → 美国住房与地产）

内置 **2 个** 四图模板（`layoutMode: 4`），文件夹 `folder-builtin-us-housing`。

| 顺序 | 模板 ID | 名称 | 何时加载 |
|------|---------|------|----------|
| ① | `builtin-us-housing-activity` | 住房 · 供需与景气 | **默认第一步**：量端（许可→销售→库存→完工） |
| ② | `builtin-us-housing-price-finance` | 住房 · 价格与融资 | 价端：房价→利率→自有率→信用 |

### 模板 ① — 住房 · 供需与景气

| 图 | slotTitle | 序列（显示名） |
|----|-----------|----------------|
| 1 | L1 领先：许可 vs 单户开工 | 建筑许可 同比、单户新屋开工 同比 |
| 2 | L2 销售：新屋销售 | 新屋销售 同比 |
| 3 | L3 库存：新屋可售月数 | 新屋可售月数（<4 紧俏、>6 过剩） |
| 4 | L4 完工：住房完工 | 住房完工 同比 |

**分析顺序**：许可转负=见顶最早信号 → 新屋销售定需求方向 → 库存月数跳升先于开工下滑 → 完工滞后释放产能。

### 模板 ② — 住房 · 价格与融资

| 图 | slotTitle | 序列（显示名） |
|----|-----------|----------------|
| 1 | L5 房价：Case-Shiller | Case-Shiller 全国房价 同比 |
| 2 | L6 融资：30Y vs 15Y 抵押利率 | 30Y 抵押利率（月均）、15Y 抵押利率（月均） |
| 3 | L7 自有住房率 | 自有住房率（%，季） |
| 4 | L8 信用：单户抵押贷款拖欠率 | 单户住宅抵押贷款拖欠率（%，季） |

**分析顺序**：房价同比转负是深度调整信号 → 抵押利率压制需求（对照 ① 图 2）→ 自有率见顶反映可负担性恶化 → 拖欠率抬头=信用周期下行确认。

## 决策树

| 观察 | 对照图位 | 典型结论 |
|------|----------|----------|
| 许可同比转负 + 库存月数跳升 | ①1 + ①3 | 周期见顶，开工/价格承压 |
| 抵押利率高位 + 新屋销售走弱 | ②2 + ①2 | 利率压制生效，需求收缩 |
| 房价同比放缓 + 完工高位 | ②1 + ①4 | 供给释放叠加需求弱，房价下行风险 |
| 拖欠率抬头 + 自有率回落 | ②4 + ②3 | 地产信用周期下行 |
| 许可回升 + 利率见顶回落 | ①1 + ②2 | 周期触底，地产先于经济复苏 |

## 变动率与时间轴

- DB 存**水平值**；变换由 `seriesCalcConfigMap` 计算。
- 月频流量（PERMIT/HOUST1F/HSN1F/COMPUTSA）：`op: "yoy"`, `frequency: "month"`。
- MSACSR（月数）、RHORUSQ156N/DRSFRMACBS（季频率值）：`op: "none"`。
- CSUSHPINSA（房价指数月频）：`op: "yoy"`。
- 抵押利率（周频）：`op: "none"` + `frequency: "month"` + `avg` 月均对齐。

## 关于成屋销售（EXHOSLUSM495S）

NAR 授权，FRED API 仅回约 1 年（~13 期），同比几乎无法成图。已入库、挂 `us.nar.existing_home_sales` 包并**持续累积**，但**暂不进默认模板**（数据攒够 ≥2 年后再纳入模板 ① 图 2 与新屋销售对比）。当前可在目录中自选查看。

## 每期 checklist

**模板 ①（月度新建住宅/新屋销售发布后）**：许可同比方向 → 新屋销售同比 → 库存月数 → 完工同比。
**模板 ②（Case-Shiller/PMMS/季度数据更新后）**：房价同比 → 30Y 抵押利率 → 自有率 → 拖欠率斜率。
