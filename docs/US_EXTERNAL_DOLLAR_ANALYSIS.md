## 核心问题（L0）

> 美元处于强势还是弱势周期？贸易逆差是在扩大还是收窄？外部融资与净国际头寸是否可持续？汇率变动如何通过贸易量与贸易条件影响增长与通胀？

## 与其他模板分工

| 主题 | 归属 | 本框架 |

## 决策树

| 观察 | 对照图位 | 典型结论 |
|------|----------|----------|
| 广义美元上行 + 出口同比转弱 | ①1 + ①4 | 汇率压制外需 |
| 逆差扩大且进口同比主导 | ①3 + ①4 | 内需/库存驱动逆差 |
| EME 美元指数单独走强 | ①2 | 新兴侧美元压力 |
| 经常账户逆差扩大 + NIIP 恶化 | ②1 + ②2 | 外部融资依赖上升 |
| 进口价格同比上行 + 贸易条件恶化 | ②3 + ②4 | 进口成本冲击、滞胀风险 |

## 变动率与时间轴

- DB 存**水平值**；变换由 `seriesCalcConfigMap` 计算。
- 日频汇率（DTWEX*）：`op: "none"` + `frequency: "month"` + `avg` 月均。
- 月频流量（BOPTEXP/BOPTIMP）与价格指数（IQ/IR）：`op: "yoy"`, `frequency: "month"`。
- 贸易差额、经常账户、NIIP、贸易条件：`op: "none"`（水平值）。

## 每期 checklist

1. 广义美元月均方向是否变化？AFE/EME 是否背离？
2. 贸易差额与进出口同比：逆差变化由出口还是进口主导？
3. 经常账户与 NIIP：流量与存量是否同向恶化？
4. 进出口价格同比与贸易条件：是否出现进口成本冲击？
5. 与 Overview 实际进出口、通胀域进口通道交叉验证。
